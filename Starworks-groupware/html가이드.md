
### 루트 `html` 
*   `menu_user_view(X).html`: 사용자 메뉴 뷰로, 플레이스홀더 또는 이전 버전

### `approval`  (결재 관련)
*   `approval-report-view.html`: 결재 보기
*   `approval-main.html`: 메인 결재 페이지
*   `approval-expense-view.html.html`: 결재 보기
*   `approval-expense-create.html`: 결재 생성
*   `approval-drafts.html`: 결재 목록
*   `approval-draft-create.html`: 결재 생성
*   `approval-archive.html`: 결재 보관함

### `attendance`  (근태 관련)
*   `attendance-main.html`: 메인 근태 페이지

### `calendar`  (캘린더 관련)
*   `calendar-team.html`: 팀 캘린더
*   `calendar-depart.html`: 부서 캘린더

### `community`  (커뮤니티 관련)
*   `community-notice.html`: 커뮤니티 공지사항
*   `community-board.html`: 일반 커뮤니티 게시판
*   `community-board-detail.html`: 커뮤니티 게시판 상세 보기
*   `community-board-create.html`: 커뮤니티 게시판 글쓰기

### `document`  (문서 관련)
*   `document_upload.html`: 문서 업로드
*   `document_list.html`: 문서 목록

### `main`  (메인 페이지 관련)
*   `user-main-v2.html`: 사용자 메인 페이지 버전 2
*   `user-main-v1.html`: 사용자 메인 페이지 버전 1

### `orgchart`  (조직도 관련)
*   `orgchart-user-v2.html`: 사용자 조직도 버전 2
*   `orgchart-user-v1.html`: 사용자 조직도 버전 1

### `project` (프로젝트 관련)
*   `project-myproject.html`: 내 프로젝트 보기
*   `project-main.html`: 메인 프로젝트 페이지
*   `project-list.html`: 프로젝트 목록
*   `project-edit.html`: 프로젝트 편집
*   `project-detail.html`: 프로젝트 상세 보기
*   `project-create.html`: 새 프로젝트 생성
*   `project-archive.html`: 프로젝트 보관함
